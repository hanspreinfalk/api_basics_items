Game Items API Documentation

API Endpoints

Items

Method Endpoint Description Request Body
POST /api/items Create new item(s) { id, name, type, effect }
GET /api/items Get all items -
GET /api/items/:id Get item by ID -
PUT /api/items/:id Update item { name?, type?, effect? }
DELETE /api/items/:id Delete item -
